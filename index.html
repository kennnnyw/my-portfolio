Doomsday

  • November 2017 - April 2018
  • University Project

An Android application to facilitate the teaching and learning of John Conway's Doomsday algorithm for calculating the weekday for a given date. The app features guides, quizzes, push notifications and stat tracking. This application was developed for my final year project at University.

  • Java
  • JSON
  • Data Persistence
  • UX/UI
  • BitBucket
  • Sketch
  • Invision

Hygiene Ratings

  • February 2018

An iOS application for viewing the hygiene ratings of UK businesses. The app makes use of geolocation to provide information about nearby eateries and communicates with a RESTful web service to provide search functionality.

Clear Weather
